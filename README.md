# 2026年建設現場入場基準対応・損害保険提案DM

建設業の経営者向けに、2026年から施行される建設現場入場基準の変更と、それに対応するための損害保険提案を行うダイレクトメール（DM）用HTMLドキュメントです。

## 📄 概要

このプロジェクトは、建設業許可を取得した企業向けのマーケティング資料です。2026年から厳格化される建設現場への入場基準について説明し、必要な損害保険への加入を提案します。

## 📋 ドキュメント構成（全5ページ）

| ページ | 内容 |
|--------|------|
| 1ページ目 | **表紙** - 2026年現場入場基準変更の導入・問題提起 |
| 2ページ目 | **問題提起+データ** - 法令・基準の変更点、建設業界の現状データ |
| 3ページ目 | **保険説明①②** - 建設工事保険・賠償責任保険の説明と事故事例 |
| 4ページ目 | **保険説明③+まとめ** - 労災上乗せ保険、3つの保険セットの重要性 |
| 5ページ目 | **行動喚起** - 準備タイムライン、導入ステップ、連絡先・LINE QR |

## 🚀 使用方法

### ブラウザで表示

```bash

open construction-site-entry-standards-proposal.html
```

### 印刷する場合

1. ブラウザでHTMLファイルを開く
2. `Ctrl+P`（Windows）または `Cmd+P`（Mac）で印刷ダイアログを開く
3. 用紙サイズ「A4」、余白「なし」を選択
4. 「背景のグラフィック」をオンにして印刷

## 📝 提案する保険商品

1. **建設工事保険** - 工事中の事故・損害をカバー
2. **賠償責任保険** - 対人・対物事故の賠償責任をカバー
3. **労災上乗せ保険** - 従業員の安全・労災補償を強化
